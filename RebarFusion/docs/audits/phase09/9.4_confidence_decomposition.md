# Phase 9.4 — Confidence Decomposition

Scope: `core/engineering/confidence.py` (new), wired into `run_phase9.py`, run against `SS-GF-01(M).dxf`'s 3 current families. Last major audit before freezing Phase 9, per plan.

## Design
`ConfidenceBreakdown` with 7 independent dimensions, each scored by a function consuming only its own owning phase's data:

| Dimension | Owning phase | Source data |
|---|---|---|
| `geometry` | 2/3/6 | Member bbox/length well-formedness sanity check |
| `recognition` | 7 | Mean Phase 7 evidence-based recognizer confidence |
| `plausibility` | 7.6 | Phase 7.6 decision (accept=1.0/review=0.6/reject=0.0) |
| `annotation` | 8 | Best direct `TOKEN_MARK` candidate score for the representative |
| `association` | 8 | Mean of all candidate scores (mark/diameter/spacing/etc.) for the representative |
| `spacing` | 9.3 | `EngineeringFamily.spacing_confidence` (reused, not recomputed) |
| `family_consistency` | 9 | Mean per-member orientation/length agreement from `_discover_members` |

`overall` is the **geometric mean**, not arithmetic — deliberately penalizes one very weak dimension more than an average would, matching "a family is only as trustworthy as its weakest stage." `weakest_dimension` names the specific stage responsible, not just the number.

## Outputs
`debug/phase09_4/<drawing>/`: `confidence_breakdown.json` (full per-family breakdown with evidence), `confidence_summary.json` (mean overall, weakest-dimension counts), `confidence_histogram.json`, `low_confidence_objects.json` (overall < 0.70, with weakest dimension + evidence), and `family_confidence_gallery/<mark>_<uuid8>.png` — one bar-chart card per family, color-coded (green ≥0.8, yellow ≥0.5, red below), showing exactly which dimension to look at.

## The decomposition immediately validates itself
N4's `plausibility: 0.60` is not a new finding — it's the confidence system correctly re-surfacing the same 5.34mm fragment cluster that Phase 7.6 flagged as "review" and Phase 9.1/9.2/9.3 independently kept rediscovering, now quantified as a specific, named dimension score rather than buried in one opaque 0.276 average. This is the fourth independent method converging on the same signal (Phase 7.6 statistical, Phase 9.2 family-comparison, Phase 9.3 spacing-gap, now Phase 9.4 confidence-dimension) — each one arriving at it through completely different logic. That consistency is itself evidence the decomposition is measuring something real, not noise.

N4's `overall: 0.75` is notably **more informative and less alarming** than the old pooled "Average Confidence: 0.212" — the old number averaged across all families and conflated every dimension into one figure, making a genuinely-fine family (N6 at 0.93) and a genuinely-uncertain one (N4 at 0.75, still not "broken") indistinguishable in the summary. The decomposition shows N4 is specifically weak in `annotation`/`association` (its mark-association scores were middling, 0.52-0.57 — consistent with it being one of the components affected by the review-tier fragment classification), not uniformly bad.
